# 简介
本项目是采用 Vu3 + Typescript + Vite 方式搭建的 admin 框架，代码多采用 setup 方言的形式写成。推荐采用 vscode + volar 插件进行开发。这样可以获得很好地 TS 体验

# 提供的建议功能
1. 登陆
2. Layout 布局
3. 富文本
4. echarts
5. 递归菜单
6. 路由拦截
7. 集成了 router + vuex
8. 国际化支持（i18n）
